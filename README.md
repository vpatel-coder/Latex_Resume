# LaTeX Resume Template

This repository contains a reusable LaTeX resume template that you can clone, edit, and compile into a professional-looking resume.

## Features

- Clean, ATS-friendly layout
- Simple structure with clearly labeled sections
- Easy to customize (education, experience, skills, projects, etc.)
- Works with common LaTeX tools (TeX Live, MikTeX, Overleaf)



## Files

- `resume.tex` – main LaTeX source file (edit this with your own details)
- `resume.pdf` – compiled example output
- `.gitignore` – ignores LaTeX build artifacts


## How to Use

### Option 1 — Online (Overleaf)

1. Download this repository as a ZIP (`Code` → `Download ZIP`).
2. Go to Overleaf and create a new project.
3. Upload `resume.tex` (and any required `.sty` files if you add them).
4. Edit the template with your own information.
5. Recompile to generate your resume PDF.
